Dessa Script för Bash, Powershell och Python är en del av min utbildning till IT-Säkerhetsutvecklare. 
Syftet med denna uppgift är att skapa mindre automationsprogram inom IT-säkerhetsmoment som exempelvis logginsamling och analys. 
För kursens innehåll skall vi kunna läsa och förstå oss på mindre skript med stöd av AI.  Kursen är på 6 veckor där vi har fått teoretisk introduktion till samtliga skript med 2 veckor för vardera skript. 
Dessa skript är grunden till automationen och skall inte ses som fullt säkra skript eftersom dessa bör byggas vidare med extra säkerhet för att de skall vara säkra för användning inom organisationen.
